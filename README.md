
# About

此项目是 vue + element-ui 构建的后台管理系统，是后台项目 的管理系统，所有的数据都是从服务器实时获取的真实数据，具有真实的注册、登陆、管理数据、权限验证等功能。





## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + less + element-ui + axios
